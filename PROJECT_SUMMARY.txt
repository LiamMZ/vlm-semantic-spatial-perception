QUICK START
================================================================================

1. Install dependencies:
   pip install -r requirements.txt
   pip install -e .

2. Configure API keys:
   cp .env.example .env
   # Edit .env with your GEMINI_API_KEY

3. Test camera:
   python -c "from src.camera import WebcamCamera; 
              camera = WebcamCamera(); 
              frame = camera.capture_frame(); 
              print(f'Captured: {frame.shape}')"

4. Test task parsing:
   python -c "from src.task import TaskParser; 
              parser = TaskParser(); 
              result = parser.parse('pick up the red cup'); 
              print(result)"
